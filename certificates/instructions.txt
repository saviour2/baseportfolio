CERTIFICATE MANAGEMENT INSTRUCTIONS
=====================================

üìã QUICK START GUIDE
---------------------
This folder is part of the dynamic certificate management system.
Follow these simple steps to add new certificates to your portfolio.

üéØ STEP-BY-STEP PROCESS
-----------------------

STEP 1: Prepare Your Certificate Image
--------------------------------------
- Save your certificate image in this folder
- Supported formats: PNG, JPG, JPEG, WebP
- Recommended size: Under 2MB
- Recommended dimensions: 1200x900px (or similar ratio)
- Use descriptive names: aws-saa.png, azure-devops.png, etc.

STEP 2: Update the JSON Data File
----------------------------------
- Open: data/certificates.json
- Find the appropriate category:
  * "certifications" - Official certifications with credentials
  * "achievements" - Personal accomplishments and projects
  * "workshops" - Training sessions and bootcamps
  * "seminars" - Industry events and conferences
  * "events" - Community activities and learning paths

STEP 3: Add Your Entry
----------------------
Copy this template and modify it:

{
  "id": [NEXT_NUMBER],
  "title": "Your Certificate Title",
  "issuer": "Certification Body or Organization",
  "date": "2025",
  "description": "Brief description of what this certificate demonstrates or what you learned.",
  "icon": "technology-icon.svg",
  "category": "cloud",
  "tags": ["Tag1", "Tag2", "Tag3"],
  "credentialId": "CERT-123456",
  "verificationUrl": "https://verification-link.com",
  "image": "certificates/your-image-name.png"
}

üìù FIELD EXPLANATIONS
----------------------

REQUIRED FIELDS:
- id: Unique number (use next available number)
- title: Certificate or achievement name
- issuer: Who issued it (AWS, Microsoft, etc.)
- date: Year obtained
- description: What it proves/demonstrates
- icon: Icon file from icons/light/ folder
- category: cloud, devops, workshop, seminar, event
- tags: Array of relevant keywords

OPTIONAL FIELDS:
- credentialId: Official credential number
- verificationUrl: Link to verify certificate online
- image: Path to certificate image in this folder

üîß TROUBLESHOOTING
------------------

PROBLEM: Certificate doesn't appear
SOLUTION: Check JSON syntax with online validator

PROBLEM: Image doesn't load
SOLUTION: Verify image path and file exists in certificates/ folder

PROBLEM: Filter not working
SOLUTION: Ensure "category" field matches available filters

PROBLEM: Search not finding certificate
SOLUTION: Check "tags" array includes searchable keywords

üöÄ AFTER ADDING
----------------
1. Save the certificates.json file
2. Refresh your portfolio website
3. Navigate to Achievements page
4. Your new certificate appears automatically!
5. Test search and filters to ensure it works
